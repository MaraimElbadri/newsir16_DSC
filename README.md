# newsir_DSC

## Requirements
* Python 2.7 - check using this command `python --version`
* pydavid - install `pip install pyldavis`
* Bokeh - install `pip install bokeh` 
* NLTK
* scikit learn


## Features 
- [x] preprocessing 
- [x] tfidf 
- [x] clustering
- [x] topic modelling
- [x] visualization
